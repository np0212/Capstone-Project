Help Topics by Page

Home

  • This page has general quiz info for the selected course
  • The assign button will make the quiz available to your student list
  • The delete button will delete the quiz, but not the problems

Create Problems

  • Use this page to create quiz problems
  • All fields must be filled in
  • Use '$' symbols to add
  • Add Keywords you'd like to search by in the My Problems Page
  • Functions to edit and delete problems have been added

Problem Database

  • Displays all the problems in the database
  • Allows teachers to utilize keyword search
  • Allows teachers to copy problems into the selected quiz

Try My Quiz

  • You can test your quizzes here
  • This quiz view should match the student view
  • Once complete, your grade will be displayed
  • Editing Grades is not yet enabled

My Grades

  • Any quizzes completed by students will be displayed here
  • Clicking on the quiz will show the result
  • Editing Grades is not yet enabled

My Students

  • Two lists are displayed: Your Students and All Students
  • Click to Add and Remove Students from your course

My Courses

  • Courses you have created are displayed here
  • There are options to delete and rename
